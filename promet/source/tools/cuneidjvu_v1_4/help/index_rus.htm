CuneiDjVu

Руководство пользователя

CuneiDjVu - это экспериментальная программа, предназначенная для создания распознанного текстового слоя (OCR) в DjVu-файле посредством CuneiForm.

CuneiDjVu имеет лицензию "GPL 2 и выше" и полностью открытые исходные коды.

Программа работает как под Win98, так и под Win2000/XP.

CuneiDjVu использует бесплатную программную графическую библиотеку FreeImage (под лицензией GPL).

ВНИМАНИЕ:

CuneiDjVu является графической оболочкой к консольной программе СuneiForm-Linux v1.1 (собранной под Windows). Поэтому использовать CuneiDjVu для реального распознавания (создания OCR-слоя) DjVu-файлов КРАЙНЕ не рекомендуется!!! (ввиду низкого качества распознавания). Для "настоящего" распознавания DjVu-файлов используйте коммерческие OCR-продукты.

CuneiDjVu - это пока что лишь "игрушка", и не более того.

Использование:

1. Запустите файл CuneiDjVu.exe.

2. Просматривая окно программы сверху вниз, заполните все необходимые поля ввода.

Основные опции:

Язык OCR - язык распознавания. Значение по умолчанию: Русско-английский (смешанный англо-русский текст). Правильный выбор языка распознавания определяет правильность работы программы.

Детализация OCR - уровень координатной детализации OCR-данных. Значение по умолчанию: Слово. Альтернативное значение: Символ.

Режим распознавания - режим распознавания. Возможные значения: Одна колонка, Матричный принтер, Факс. Можно указать несколько значений.

4. Нажмите на кнопку Пуск - начнётся процесс распознавания выбранного DjVu-файла. Дождитесь завершения этого процесса.

В случае успешного распознавания выбранного DjVu-файла его распознанная копия будет помещена в заданную папку вывода (по умолчанию - "Рабочий Стол") с именем по умолчанию "CuneiDjVu Result" (это имя можно задать в окне по кнпке Опции в поле ввода Имя готового OCR-DjVu файла:).

Подробности:

Файл CuneiDjVu.exe - это просто GUI (визуальная оболочка), управляющий соответствующими консольными утилитами.

Пакет содержит следующие рабочие утилиты:

- djvudumpm.exe - консольная утилита для получения информации о внутренней структуре DjVu-файла. Версия: 1.2. Представляет из себя утилиту djvudump.exe http://djvu.sourceforge.net/ с добавленной возможностью выгрузки информации в текстовый файл.

- fi_ddjvu.exe - консольная утилита для декодирования любых DjVu-файлов: одно- и многостраничных; содержащих чёрно-белые, серые и цветные изображения (при декодировании автоматически создаются файлы соответствующей битовой разрядности). Версия: 1.4. Представляет из себя утилиту ddjvu http://djvu.sourceforge.net/ с добавленной поддержкой FreeImage.

- cuneiform.exe - консольная программа СuneiForm-Linux v1.1 (скомпилированная под Windows в MS VC++ 6.0).

- hocr2djvuxml.exe - консольная утилита для преобразования файлов в формате CuneiForm-hOCR в файлы в формате DjVu XML.

- djvuxmlparser.exe - стандартная консольная DjVuLibre-утилита для внедрения OCR-слоя (в виде DjVu XML-файла) в DjVu.

В папке cuneiform находится консольная программа СuneiForm-Linux v1.1 (скомпилированная под Windows в MS VC++ 6.0) со всеми необходимыми для её работы файлами. Содержимое папки cuneiform может использоваться полностью автономно от CuneiDjVu - для произвольных задач распознавания (в качестве входного формата поддерживается только 1-битный BMP).

Опции

Имя готового OCR-DjVu файла: - имя, которое будет присваиваться распознанному DjVu-файлу.

Выводить на Рабочий Стол - автоматически выбирать Рабочий Стол в качестве папки вывода - при вводе DjVu-файла, подлежащего распознаванию.

Открыть OCR-DjVu файл после обработки - открыть распознанный DjVu-файл сразу после создания в просмотрщике, ассоциированном с форматом DjVu.

Language - переключатель языка интерфейса.

Примечания

1. CuneiDjVu является полностью и абсолютно легальной программой во всех отношениях (что является её главным преимуществом). Её лицензионный статус - "свободно-бесплатная программа с открытыми исходными кодами".

2. Язык интерфейса определяется автоматически при самом первом открытии программы.
