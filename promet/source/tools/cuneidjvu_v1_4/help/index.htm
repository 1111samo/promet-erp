CuneiDjVu

Reference manual

CuneiDjVu is an experimental program to create an OCR text layer in DjVu files using CuneiForm (Russian open-source OCR program).

CuneiDjVu is licensed under "GPL 2 and later" and comes totally open-sourced.

The program works both under Win98 and Win2000/XP.

CuneiDjVu uses the free raster image program library FreeImage (licensed under "GPL 2 and later").

WARNING:

CuneiDjVu is simply a GUI for the console program ÑuneiForm-Linux v1.1 (built under Windows). That is why it is HIGHLY recommended NOT (!!!) to use CuneiDjVu for the real DjVu OCR creation purposes (due to the low OCR quality level). For the real DjVu OCR use the commercial OCR products.

CuneiDjVu is yet simply a "toy"- not more than that.

Usage:

1. Launch the file CuneiDjVu.exe.

2. Looking through the program's window (from up to down) fill all the necessary input fields.

Main options:

OCR Language - the OCR recognition language (the language to be recognized). The default value: Russian-English (a mixed Russian-English text). The correct choice of the OCR language determines the rightful OCR layer creation.

OCR Detalization - the level of the coordinate data detalization in the created OCR layer. The default value: Word. The alternative value: Character.

Recognition mode - the text layout mode on the recognized page. The possible values: Single column, Dot matrix, Fax. You can set more than one value at once.

3. Press the Process button - it will start the recognition process of the chosen DjVu file. Wait until this process is finished.

In case of the successful recognition of the chosen DjVu file its OCRed copy will be automatically placed into the destined output folder. The output OCRed DjVu file will be named by default "CuneiDjVu Result" (its name can be set in the input field Output OCRed DjVu filename: in the Options window).

Details:

The file CuneiDjVu.exe is simply a GUI (the visual wrapping) controlling the corresponding console utilities. The program package contains the following working console utilities:

- djvudumpm.exe - the console utility for getting the information about the internal DjVu file structure. Version: 1.2. It is actually a djvudump.exe utility http://djvu.sourceforge.net/ with the added feature of the text file output export.

- fi_ddjvu.exe - the console utility for decoding DjVu files. Version: 1.4. It is actually a ddjvu http://djvu.sourceforge.net/ utility with the added FreeImage support.

- cuneiform.exe - the console program ÑuneiForm-Linux v1.1 (compiled under Windows in MS VC++ 6.0).

- hocr2djvuxml.exe - the console utility for the conversion of the CuneiForm-hOCR files to the DjVu XML format.

- djvuxmlparser.exe - the standart DjVuLibre console utility for inserting the OCR-layer (prepared as a DjVu XML file) into DjVu.

The folder cuneiform contains the console program ÑuneiForm-Linux v1.1 (compiled under Windows in MS VC++ 6.0) with all the necessary (for its work) dependant files. The content of the cuneiform folder can be used totally independently from CuneiDjVu - for whatever OCR tasks (only 1-bit BMP is supported as the input format).

Options

Output OCRed DjVu filename: - the name for the resulting OCR DjVu file.

Output to Desktop - choose automatically Desktop as the output folder when inputing a DjVu file.

Open OCRed DjVu file after processing - open the OCRed DjVu file (right after the creation) in a DjVu associated viewer.

Language - the interface language switch.

Notes

1. CuneiDjVu is an absolutely free and legal program - in any senses (that is its main advantage). It license status is "a freeware open-sourced program".

2. The interface language is determined automatically at the very first program launch.
